# Nutrition & Workouts

A simple website to track daily calories, choose a nutrition/diet plan, and get
matching workout suggestions. Built with plain HTML, CSS, and JavaScript — no
build tools or installation required.

## Features

- **Diet Plans** — pick from Balanced, Low-Carb, High-Protein, or Plant-Based.
  Choosing a plan sets your daily calorie goal.
- **Calorie Tracker** — log foods with their calories and see how much of your
  daily goal remains.
- **Workouts** — suggested workouts based on your selected plan.
- Data is saved in your browser (localStorage), so it stays between visits.

## How to run

Just open `index.html` in any web browser. That's it.

## Project structure

| File         | Purpose                              |
|--------------|--------------------------------------|
| `index.html` | Page structure and content           |
| `styles.css` | All styling                          |
| `app.js`     | App logic (plans, tracker, workouts) |
